# Suggested Commands

## Run Locally
Use a local static server to ensure `fetch` API works correctly.

- **Python**:
  ```sh
  python -m http.server 8000
  ```
  Then open [http://localhost:8000/](http://localhost:8000/)

- **Node**:
  ```sh
  npx serve . -l 8000
  ```

## Testing
- No automated tests.
- **Manual Verification**:
  - Open local server.
  - Check Mobile/PC responsiveness.
  - Verify External links (Instagram, X).
  - Test contact form validation and submission.
  - Toggle theme (Dark/Light) to ensure no flickering.

## Utils
- `git status`, `git diff` for checking changes.
